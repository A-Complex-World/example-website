# A Complex World — Example Website

This repository contains the source code for the [A Complex World](https://a-complex-world.de) example/demo website, deployed via Vercel.

## 🌐 Live URL

**[https://example-website-a-complex-world.vercel.app](https://example-website-a-complex-world.vercel.app)**

## 📁 Structure

```
/
└── index.html    # Single-page website (dark theme, fully self-contained)
```

## ✨ Features

- Dark theme with animated gradient blobs
- **Inter** + **Space Grotesk** typography (Google Fonts)
- Hero section with tagline
- 6-service grid: Softwareentwicklung, Webdesign & UX, Technisches Consulting, KI-Strategie, Hosting & Infrastruktur, Mobile Apps
- Product showcases: **Neon Prophecy** (App) & **Planeria** (Management-Software)
- Stats row
- Contact CTA
- Fully responsive (mobile-friendly)
- No dependencies, no build step — pure HTML/CSS

## 🚀 Deployment

Deployed automatically via [Vercel](https://vercel.com). Every push to `main` triggers a new deployment.
